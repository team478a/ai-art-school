# v5.4.8 リッチメニュー設定

## 目的

クライアントを切り替えた状態で、管理者がそのクライアント専用のLINEリッチメニューを作成・保存・同期できるようにします。

## 作成場所

サイドメニューの「LINE・制作」内にある「リッチメニュー設定」から作成します。

クライアント別設定画面の「リッチメニュー」欄は、作成済みの `richmenu-...` IDを確認または手入力する場所です。通常はリッチメニュー設定画面で作成すると自動保存されます。

## 入力方法

- 動作が「メッセージ送信」の場合は、送信テキストに `生成`、`マイページ`、`使い方` などを入力します。
- 動作が「URLを開く」の場合は、URLに `/liff/shop?from=richmenu`、`/liff/calendar?from=richmenu`、`/liff/gacha?from=richmenu` などを入力します。
- 予約を使わないクライアントは、予約ボタンを生成・購入・マイページ・ガチャなどへ置き換えます。

## 選択式になった項目

クライアント別設定では、選択肢がある項目をプルダウンで選べるようにしました。

- 運用タイプ
- 画像生成エンジン
- 画像品質
- 承認方式
- 参加・生成条件
- 生成申請の受付方式
- 生成可能曜日

## 運用手順

1. クライアント管理で対象クライアントを「このクライアントで操作」に切り替えます。
2. クライアント別設定で、LINE Channel Secret、Channel Access Token、LIFF IDを保存します。
3. リッチメニュー設定でボタン内容を保存します。
4. 「LINEリッチメニューを作成」で状態別メニューを作成します。
5. 必要に応じて「リッチメニューを同期」で既存ユーザーへ反映します。
