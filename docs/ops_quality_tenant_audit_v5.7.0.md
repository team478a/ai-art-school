# v5.7.0 運用品質・テナント監査

## 主な変更

- 生成画像の破損、重複、低品質を検査し、条件に応じて自動再生成します。
- 生成処理のハートビートとテナント別の処理記録を保存します。
- 停止した生成依頼とジョブをテナント単位で復旧できます。
- LINE、LIFF、Stripe、AI APIの設定状態を利用機能に応じて診断します。
- テナント分離、バックアップ、管理者権限、CRON、エラー状況を一画面で監査します。
- 標準テナントの既存管理者アカウントを誤って未設定扱いしないようにしました。

## 更新後の確認

1. クライアント管理から対象クライアントの「診断」を開きます。
2. 接続、生成処理、テナント分離、バックアップ、権限の各項目を確認します。
3. 停止中の依頼がある場合は、対象クライアントを確認してから復旧処理を実行します。
4. 実際のLINEアカウントで1件生成し、画像受信まで確認します。
