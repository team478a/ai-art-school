# v5.4.9 リッチメニューかんたん設定

## 目的

リッチメニューのURLや送信テキストを手入力しなくても、用途を選ぶだけで設定できるようにしました。

## 通常使うプリセット

- 画像生成: LINEに「生成」と送信します。URL入力は不要です。
- 予約: `/liff/calendar?from=richmenu` を開きます。
- 購入・会員メニュー: `/liff/shop?from=richmenu` を開きます。
- 会員情報: `/liff/shop?from=richmenu` を開きます。
- マイページ: LINEに「マイページ」と送信します。
- ガチャ: `/liff/gacha?from=richmenu` を開きます。
- 使い方: LINEに「使い方」と送信します。
- 問い合わせ: LINEに「問い合わせ」と送信します。

## 手入力が必要な場合

通常は手入力不要です。
外部サイトや特別なLIFFページを開きたい場合だけ、用途で「自由設定（URL）」を選び、URLを入力してください。

独自メッセージを送信したい場合は、用途で「自由設定（メッセージ）」を選び、送信テキストを入力してください。

## 作業手順

1. クライアントを選択します。
2. リッチメニュー設定を開きます。
3. 各ユーザー区分のボタン用途を選びます。
4. 設定を保存します。
5. LINEリッチメニューを作成します。
6. 全ユーザーへ同期します。
