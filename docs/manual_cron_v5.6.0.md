# v5.6.0 CRON代替の非同期手動処理

## 追加機能

- ダッシュボードから「手動処理を実行」できます。
- 依頼一覧から「次の1件を手動処理」できます。
- 受信済み・再試行待ち・停止した処理中依頼の詳細画面から、その依頼だけを「今すぐ手動処理」できます。
- 手動処理は、現在操作中のクライアントだけを対象にします。
- 1回の操作で画像生成は1件だけ処理し、共有サーバーのタイムアウト負荷を抑えます。
- 期限到来済みのリマインド、空席通知、参加後フォローも同時に実行します。
- ボタンを押すと管理画面へすぐ戻り、画像生成はサーバー側で継続します。
- プロンプト生成はOpenAI APIキーが設定されている場合、OpenAIを使用します。
- Claude APIはプロンプト生成経路から除外しました。旧設定や共通設定にClaude APIキーが残っていても呼び出しません。

## 安全対策

- 管理者またはオーナーだけが実行できます。
- CSRF検証を行います。
- 直近5分以内に処理中の依頼は二重実行しません。
- 停止した処理中ジョブ、失敗ジョブ、実行待ちジョブを手動でキューへ戻せます。
- テナントIDで対象データを絞り込み、別クライアントの依頼は処理しません。

## 使用方法

1. 管理画面のダッシュボードを開きます。
2. 自動処理（cron）が停止表示の場合、「手動処理を実行」を押します。
3. 依頼が複数ある場合は、完了後にもう一度押します。
4. 特定の依頼を処理したい場合は、依頼詳細の「今すぐ手動処理」を押します。

処理受付後は画面を閉じても構いません。画像生成APIの応答により数分かかる場合があります。
