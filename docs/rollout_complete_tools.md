# v5.0.3 横展開 完成ツール

## 目的

AIアート教室システムを新規クライアントへ横展開するため、オーナー向けの初期設定、検査、バックアップ、導入資料出力を一画面に集約しました。

## 追加内容

1. 新規クライアント作成ウィザード
   - クライアント名、サービス名、教室名、公開URL、会社情報、標準テンプレートをまとめて登録できます。

2. 設定テンプレート管理
   - AIアート教室標準、単発ワークショップ、月額会員型のテンプレートを用意しました。
   - 適用前に現在の設定スナップショットを保存します。

3. 公開前テスト実行ページ
   - 公開URL、利用規約、プライバシーポリシー、特商法、Stripe Webhook、LINE、LIFF、AI設定の状態を確認できます。

4. 横展開用バックアップ・復元強化
   - 従来の全体復元に加え、基本情報、公開ページ、運用フロー、料金表示、AI品質設定だけの部分復元に対応しました。

5. 本番切替チェック
   - Stripe本番キー、公開URL、販売プラン、LINE、AI、当日参加条件などを公開前に確認できます。

6. クライアント別導入ドキュメント自動生成
   - 導入ガイド、チェックリスト、引き継ぎ資料、設定JSONを出力できます。

## 運用上の注意

- 横展開はクライアント別コピー展開を前提にします。
- `config/db.php`、`config/installed.lock`、`storage/`、`uploads/` はクライアントごとに分離してください。
- LINE公式、Stripe、AI APIキーはクライアント所有を標準にしてください。
- アップデートZIPは共通本体として使い、設定、画像、予約、支払い履歴は上書きしない運用を維持してください。

## 確認URL

- 管理画面: `/admin/client-setup`
- 導入ガイド出力: `/admin/client-setup/guide`
- チェックリスト出力: `/admin/client-setup/checklist`
- 引き継ぎ資料出力: `/admin/client-setup/handover`
- 設定JSON出力: `/admin/client-setup/export`
