# v4.9.8 横展開・初期設定画面の整理

## 目的

横展開時に使う「横展開・初期設定」画面を日本語で整理し、新規クライアント導入時の確認をしやすくしました。

## 修正内容

- 横展開・初期設定画面の文字化けを解消
- 導入進捗、公開前ヘルスチェック、初期設定チェックを見やすく整理
- 標準フロー設定を説明付きで表示
- 設定JSONの読み込みフォーム名を `client_setup_json` に修正
- 設定スナップショットの保存・復元UIを整理
- 導入時チェックリストを追加

## 注意

この更新は画面表示と設定フォームの修正です。既存の予約、画像、決済、LINEユーザー情報は変更しません。
