# 横展開・初期設定チェック

## 目的

新しいクライアントへ AI アート教室システムを展開するときに、設定漏れを管理画面で確認できるようにするための更新です。

## 追加・修正内容

- `/admin/client-setup` の文字化けを解消
- 横展開用の初期設定チェックを日本語で表示
- 公開前ヘルスチェックを追加
- クライアント情報、会社情報、問い合わせ情報、標準フロー設定を管理画面で整理
- 設定JSONの書き出し、読み込み、スナップショット復元の案内を整理
- 導入時チェックリストを画面内に追加

## 横展開時に案件ごとに分離するもの

- `config/db.php`
- `config/installed.lock`
- `storage/`
- `uploads/`
- LINE公式アカウント
- 予約用 LIFF ID
- 購入用 LIFF ID
- Stripe APIキー、Price、Webhook
- AI APIキー

## 導入前確認

1. 新規ドメインとDBを用意する
2. オーナー管理者を作成する
3. LINE公式とLIFFをクライアント所有で作成する
4. Stripeの商品、Price、Webhookをクライアント所有で作成する
5. AI APIキーとフォールバック先を設定する
6. 公開ページ、利用規約、プライバシーポリシー、特商法を確認する
7. 予約、承認、当日参加、画像生成、購入、Webhook反映をテストする

## 注意

アップデートZIPでは、`config/db.php`、`config/installed.lock`、`storage/`、`uploads/` を上書きしない運用を継続してください。
