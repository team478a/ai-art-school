# v5.2.2 管理者所属クライアント設定

## 目的

SaaS運用で、オーナーが複数クライアントを管理しながら、管理者・スタッフには担当クライアントだけを扱わせるための更新です。

## 追加内容

- `admin_users.tenant_id` を追加します。
- 既存の「管理者アカウント」画面に「所属クライアント」を追加しました。
- 管理者・スタッフはログイン時に所属クライアントが自動選択されます。
- オーナーは全クライアントを扱えます。必要に応じてクライアント管理画面の「管理中にする」で対象を切り替えます。

## 運用方法

1. オーナーで管理画面にログインします。
2. 「管理者アカウント」を開きます。
3. 管理者またはスタッフを追加します。
4. 所属クライアントを選択します。
5. 対象者がログインすると、そのクライアントの管理画面として動作します。

## 注意

- オーナーは所属クライアントを持ちません。
- 管理者・スタッフの所属クライアントが未設定の場合、サーバー側の標準テナント判定に従います。
- 既存データ、画像、予約、決済履歴は上書きしません。
