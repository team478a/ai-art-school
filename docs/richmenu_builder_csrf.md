# v5.4.5 リッチメニュー設定修正

## 修正内容

- リッチメニュー同期フォームにCSRFトークンを追加しました。
- リッチメニュー生成フォームを追加し、初回・参加済み・回数券・サブスクごとのメニューをLINE側に作成できるようにしました。
- 6ボタンの表示名、短い表示文字、動作、送信テキスト、URLを管理画面から設定できるようにしました。
- 現在操作中のクライアントに紐づいた設定として保存されます。
- LINEリッチメニュー画像の描画処理を修正し、生成画像に6ボタンが正しく描かれるようにしました。

## 操作手順

1. 管理画面で対象クライアントに切り替えます。
2. `リッチメニュー設定` を開きます。
3. ボタン項目を設定して保存します。
4. 初回・参加済み・回数券・サブスクの各リッチメニューを生成します。
5. 生成されたIDが基本設定に入っていることを確認します。
6. 全ユーザー同期を実行します。

## 注意点

- LINE Channel Access Token が未設定の場合、生成と同期は失敗します。
- URLを開くボタンは、相対URLでも現在の公開ドメインから絶対URLに変換されます。
- 同期は現在操作中のクライアントのユーザーだけを対象にします。
