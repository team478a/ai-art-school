# 生成申請の受付時間設定

教室予約を使わず、指定した時間帯だけ画像生成申請を受け付けるための設定です。

## 設定場所

管理画面の「API設定」内にある「利用制限」で設定します。

## 受付方式

- 教室参加確認後のみ  
  従来どおり、予約承認後に当日参加確認をしたユーザーだけ生成できます。

- 受付時間内なら生成可能（予約不要）  
  教室予約を使わず、設定した開始時刻から終了時刻まで生成申請できます。

- 参加確認済み、または受付時間内  
  教室参加確認済みのユーザー、または受付時間内のユーザーが生成できます。

- 常時受付  
  時間や参加確認で制限せずに生成申請できます。

## 受付時間

`生成受付開始` と `生成受付終了` に `HH:MM` 形式で入力します。

例:

- 10:00 から 17:00 まで受付
- 22:00 から 02:00 まで受付

日付をまたぐ時間帯にも対応しています。

## 受付時間外メッセージ

未入力の場合は標準メッセージがLINEに表示されます。イベント名や次回受付時間を案内したい場合だけ入力してください。

## 注意

受付方式を「受付時間内なら生成可能」にした場合、日次生成数は教室スケジュールではなく、API設定の「1ユーザーあたりの1日最大依頼数」を使います。
