# v5.0.2 横展開 引き継ぎメモ書き出し

## 目的

新規クライアントへ横展開するときに、設定概要、運用フロー、確認URL、残作業をMarkdownで書き出せるようにします。

## 追加内容

- `/admin/client-setup/handover` を追加
- 「横展開・初期設定」画面に「引き継ぎメモを書き出す」ボタンを追加
- APIキー、Webhook署名、DB情報、個人情報、画像データを含めずに引き継ぎ情報を出力
- クライアント側が所有する外部サービスと、アップデート時に上書きしない領域を明記

## 使い方

1. 管理画面で「横展開・初期設定」を開く
2. クライアント情報と運用フローを保存する
3. 「引き継ぎメモを書き出す」を押す
4. ダウンロードされたMarkdownを、導入・納品・運用引き継ぎの記録として使う
