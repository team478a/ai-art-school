# v5.4.3 Tenant Proxy Menu Cleanup

## 目的

管理者がクライアント設定を代行している状態で、個人情報に近い「ユーザー」画面が表示されないようにしました。

## 変更内容

- テナント切替中のサイドメニューから「ユーザー」を非表示にしました。
- テナント切替中に `/admin/users` を直接開いた場合も、ダッシュボードへ戻します。
- テナント切替中は、通常運用と設定代行に必要な画面へ導線を絞ります。

## 代行設定で使う主な画面

- このクライアント設定
- リッチメニュー設定
- クライアント管理
- カレンダー、教室・参加管理、予約履歴、出席履歴、QRコード
- 依頼一覧、一斉メッセージ、ギャラリー、ガチャ運用
- 決済履歴、チケット履歴、キャンセル履歴、統計・出力
