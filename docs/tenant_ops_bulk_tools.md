# v5.3.4 クライアント運用一括ツール

## 目的

SaaS化に向けて、複数クライアントを管理するための運用機能をまとめて整備しました。

## 追加・改善内容

- クライアント管理画面の日本語表記を整理
- クライアントの状態に「アーカイブ」を追加
- 一覧画面から有効化、停止、アーカイブを操作可能に変更
- 新規クライアント作成後、クライアント別設定画面へ直接遷移
- クライアント別設定画面に次の設定導線を追加
- 月次利用・請求レポートにCSV出力を追加
- 引き継ぎメモの日本語表記を整理
- テナント設定項目、利用サマリー、警告文の文字化けを修正

## 運用手順

1. 「クライアント管理」から新規クライアントを追加
2. 作成後に表示される「クライアント別設定」でLINE、LIFF、Stripe、AI APIを入力
3. 「疎通チェック」でWebhook、LIFF、Stripe、公開ページを確認
4. 「バックアップ」で導入直後の状態を保存
5. 月末に「月次利用・請求レポート」からCSVを出力

## 状態の使い分け

- 有効: 通常運用中
- 停止中: 一時停止、契約確認中、未払いなど
- アーカイブ: 解約済み、過去案件、参照のみ

アーカイブは削除ではありません。履歴とデータを残したまま、通常運用対象から外すために使います。

## 注意

- 標準クライアントは停止・アーカイブできないようにしています。
- アップデートでは `config/`、`storage/`、`uploads/` の顧客データは上書きしない前提です。
- CSVはExcelで開きやすいようにUTF-8 BOM付きで出力します。
