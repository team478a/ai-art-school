# v5.2.0 Tenant Error Monitor

## 目的

SaaS運用時に、どのクライアントでエラーが起きているかを早く見つけられるようにします。

## 追加内容

クライアント管理画面に、クライアント別の障害監視サマリーを追加します。

- 直近24時間のエラー件数
- 直近7日間のエラー件数
- 最新エラーの発生元
- 最新エラーの内容
- 画像生成、自動処理、決済、システムログを横断して確認

## 重複を避けた点

既存の操作ログ画面は詳細ログ確認用として残します。
v5.2.0では新しいメニューを増やさず、オーナー向けのクライアント管理画面に要約だけを追加します。

## 安全性

途中導入の環境でも壊れないように、テーブルとカラムの存在を確認してから集計します。
存在しないテーブルは監視対象から自動的に外します。
