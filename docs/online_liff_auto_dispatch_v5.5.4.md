# v5.5.4 オンライン生成LIFF自動振り分け

オンライン生成型のクライアントで、既存の予約用LIFFを開くと予約カレンダーが表示される問題を修正しました。

## 動作

- 運用タイプが「オンライン生成型」の場合、`/liff/calendar` を開いても画像生成画面を表示します。
- 「オンライン生成を有効」「教室運営を無効」の組み合わせでも同じ動作になります。
- 画像生成用LIFF IDが未設定の場合、オンライン生成型に限り予約用LIFF IDを利用できます。
- 教室運営型とハイブリッド型では、予約カレンダーの動作を変更しません。

既存の予約用LIFFを流用する場合、LINE Developers側のEndpoint URLを変更する必要はありません。
