# v5.5.0 オンライン生成専用リッチメニュー

## 目的

予約や教室参加を使わず、曜日と時間だけで画像生成を受け付けるクライアント向けに、LINEリッチメニューを「画像生成」1ボタンだけで運用できるようにしました。

## 使い方

1. 管理画面で対象クライアントに切り替えます。
2. `リッチメニュー設定` を開きます。
3. `予約なし・オンライン生成だけで使う` の `生成ボタンだけの設定にする` を押します。
4. 各ステータスの `LINEリッチメニューを作成` を実行します。
5. 最後に `全ユーザーへ同期` を実行します。

## LINE側の動作

- リッチメニューは大きな「画像生成」ボタン1つだけになります。
- ボタンを押すとLINEに `生成` と送信されます。
- 実際に生成できるかどうかは、クライアント別設定の生成受付方式、曜日、時間、生成数で判定します。

## 注意

- 予約、参加、購入、ガチャを使うクライアントでは通常の6ボタン構成を使ってください。
- テンプレート適用後も、LINE側には既存のRich Menu IDが残っている場合があります。必ずリッチメニューを作成し直してから同期してください。
