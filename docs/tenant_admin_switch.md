# v5.2.1 Tenant Admin Switch

## 目的

SaaS運用時に、オーナーが管理画面で操作対象クライアントを明示的に切り替えられるようにします。

## 追加内容

- クライアント管理画面に「管理中にする」ボタンを追加
- 管理中クライアントをセッションに保持
- 管理画面では、セッションで選択したクライアント設定を優先
- 「自動判定に戻す」ボタンでドメイン・URL・既定クライアント判定へ戻せる

## 重複を避けた点

新しいサイドメニューや別ページは追加していません。
既存の「クライアント管理」画面に操作を統合しています。

## 既定にするとの違い

- 「管理中にする」: 自分の管理セッションだけ一時的に切り替える
- 「既定にする」: ドメインやtenant指定がない場合の全体既定クライアントを変更する

## 注意

切替は管理画面のみで有効です。
LINE Webhook、Stripe Webhook、LIFFなどの公開側入口は、これまで通り `tenant` パラメータ、専用URL、ドメインで判定します。
