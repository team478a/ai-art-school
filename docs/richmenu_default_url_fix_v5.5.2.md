# v5.5.2 リッチメニュー生成ボタン修正

## 修正内容

- 画像生成だけで使うクライアント向けに、LINEリッチメニューの生成ボタンを「メッセージ送信」ではなく「URLを開く」に変更しました。
- 生成ボタンのURLは `/liff/generate?from=richmenu` です。
- LINE公式アカウントの自動応答が「生成」メッセージに反応してしまう問題を避けます。
- リッチメニュー設定画面の文字化けを修正し、用途を選ぶとURLや送信内容が自動入力されるようにしました。
- 共通メニュー作成ボタンで、画像生成のみの1ボタンメニューをLINE側に作成し、デフォルトメニューとして反映できるようにしました。

## 運用手順

1. 管理画面で対象クライアントを選択します。
2. `リッチメニュー設定` を開きます。
3. 予約を使わない場合は `画像生成だけの設定にする` を押します。
4. `共通メニューを作成してLINEに表示` を押します。
5. スマホのLINEアプリを開き直して、下部メニューが表示されるか確認します。

## 注意

- 「設定を保存」だけではLINE側の画像は変わりません。LINE側に表示するには作成ボタンが必要です。
- 手入力で「生成」と送った場合は、LINE公式アカウント側の自動応答設定が反応することがあります。リッチメニューの生成ボタンから使う運用にしてください。
