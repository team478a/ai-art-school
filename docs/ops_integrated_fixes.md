# v4.9.9 運用改善 一括修正

## 目的

AIアート教室の運用中に出ていた表示・送信・生成品質まわりの残タスクをまとめて修正します。

## 修正内容

- ダッシュボードの文字化けを修正
- Stability AI残高更新の表示を改善
  - JSON以外の応答、権限エラー、ログイン切れの場合も画面に理由を表示
  - 複数AIの設定状態をバッジで表示
  - 自動切替の設定状態を表示
- LINE送信数のカウントを補正
  - push APIの呼び出し回数ではなく、送信したLINEメッセージ数で加算
- 画像再送の文言を修正
  - LINE画像送信が失敗した場合は、絶対URLのテキスト送信へフォールバック
  - 再送ログの文字化けを修正
- 画像生成プロンプトの品質ガードを強化
  - 日本語入力でも人物・子ども判定が効くようにキーワードを追加
  - 顔、手、子ども画像の崩れを抑えるガードを追加
  - 不気味な顔、崩れた手足、子どもの不適切表現をネガティブプロンプトに追加
- 決済完了ページを修正
  - `/liff/paid` をクリーンな専用コントローラーに差し替え
  - 真っ白画面、文字化けを回避
- リッチメニュー出し分けサービスの文字化けを修正
  - 初回、参加済み、回数券、サブスクの表示名を日本語化

## 注意

この更新はコードのみを上書きします。以下は上書きしません。

- `config/db.php`
- `config/installed.lock`
- `storage/`
- `uploads/`

## 今後の候補

- AIごとの残高、上限、切替履歴の一覧表示
- 画像生成結果の評価ボタン
- NG画像を学習用に分類する管理画面
- 顔・手の崩れが多い入力パターンの分析
