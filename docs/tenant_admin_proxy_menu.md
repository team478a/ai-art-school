# v5.4.2 Tenant Admin Proxy Menu

## 目的

SaaS運用で管理者がクライアントの設定を代行する際、テナント切替後のサイドメニューから必要な設定画面へ直接移動できるようにしました。

## 変更内容

- テナント切替中のサイドメニューに「このクライアント設定」を表示します。
- 「このクライアント設定」は選択中テナントの `/admin/tenants/{id}/settings` に直接移動します。
- サイドメニューに「リッチメニュー設定」を追加しました。
- リッチメニュー設定は現在操作中のテナント設定を参照するため、クライアントごとのLINEリッチメニュー出し分けを確認・同期できます。

## 管理者が代行設定するときに見る場所

1. クライアント管理
2. このクライアント設定
3. リッチメニュー設定
4. LINE設定、公開ページ設定、API設定

テナント切替中は、通常運用メニューと代行設定に必要なオーナー向けメニューだけを残します。
