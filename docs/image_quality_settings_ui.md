# v4.9.7 画像品質設定UI

## 目的

管理画面のAPI設定から、画像生成の品質方針とエンジン切替を操作できるようにしました。

## 追加した設定

- 人物・子ども向け安全エンジン
- 高品質生成エンジン
- 画像品質方針
- OpenAI画像品質
- Stability残高不足時の自動切替
- Stabilityフォールバック先
- 禁止ワード・避けたい表現

## 推奨設定

顔や手の崩れを減らしたい場合は、人物・子ども向け安全エンジンをOpenAIにしてください。
Stability AIの残高不足や一時的なAPIエラーに備え、フォールバック先もOpenAIにすることを推奨します。
