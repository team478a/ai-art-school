# v5.3.5 クライアント追加・SaaS運用仕上げ

## 目的

新しいクライアントを追加したあと、初期設定、導入チェック、月次確認、バックアップまで管理画面上で迷わず進められる状態にします。

## 主な変更

- クライアント管理画面の説明文と操作導線を整理
- クライアント追加フォームを日本語化
- クライアント別設定画面を日本語化
- 導入チェック画面を日本語化
- クライアント別バックアップ画面を日本語化
- 月次利用・請求レポート画面を日本語化
- 新規クライアント導入時に確認する項目を画面内に明記

## 導入後の確認

1. `/admin/tenants` でクライアント一覧が表示される
2. 新規クライアントを追加できる
3. クライアント別設定を保存できる
4. 導入チェックで不足項目が確認できる
5. クライアント別バックアップを作成できる
6. 月次利用・請求レポートを確認できる

## 注意

既存の予約、画像、決済、LINE履歴、アップロード済みファイルは上書きしません。
`config/db.php`、`config/installed.lock`、`storage/`、`uploads/` は更新対象外です。
