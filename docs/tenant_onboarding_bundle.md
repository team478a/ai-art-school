# v5.2.6 クライアント追加オンボーディング統合

## 目的

SaaS化・横展開で新しいクライアントを追加するときに、台帳登録、よく使う初期設定、初期管理者作成を同じ画面で進められるようにしました。

## 追加内容

- クライアント追加画面に、LINE、LIFF、Stripe、生成制限、運用ルールの初期入力欄を追加
- クライアント追加画面に、初期管理者またはスタッフの作成欄を追加
- 初期管理者のメール重複、メール形式、8文字未満パスワードを保存前に検証
- クライアント作成後は、対象クライアントの「クライアント別設定」へ遷移
- 作成直後の設定確認、初期設定チェック、引き継ぎメモ出力につなげやすく改善

## 運用メモ

1. オーナーが「クライアント管理」から新規クライアントを追加します。
2. 分かる範囲で LINE、LIFF、Stripe Price ID、生成制限、運用ルールを入力します。
3. 必要に応じて、クライアント担当者の初期管理者アカウントを同時に作成します。
4. 保存後、クライアント別設定画面で不足項目を確認します。
5. 初期設定チェックと引き継ぎメモを使って、LINE Developers、Stripe、AI API、公開ページ設定を完了します。

## 注意

- オーナー権限はこの画面では作成しません。システム管理者のみが保持します。
- 初期管理者欄を空欄にすれば、アカウント作成なしでクライアントだけ追加できます。
- シークレットキーやWebhook Secretなどの機密情報は、作成後のクライアント別設定画面で登録してください。
