# オンライン生成受付設定 v5.4.7

## 設定場所

標準アカウントは `API設定` → `利用制限` で設定します。

クライアント操作中、または新規クライアントごとの設定は `クライアント管理` → 対象クライアントの `設定`、または左メニューの `このクライアント設定` → `運用ルール` で設定します。

クライアント操作中は標準アカウント用の `API設定` は表示しません。

## 予約なしでオンライン生成を使う場合

1. `運用タイプ` を `オンライン生成型` にします。
2. `オンライン生成を有効にする` をオンにします。
3. `生成申請の受付方式` を `受付時間内なら生成可能` または `常時受付` にします。
4. 必要に応じて生成可能開始日、終了日、曜日、受付開始時刻、受付終了時刻を設定します。
5. `期間内の1ユーザー最大生成依頼数` を設定します。
6. リッチメニューに生成ボタンと購入ボタンを設定します。

## 主な項目

- オンライン生成を有効にする
- 生成申請の受付方式
- 生成可能開始日
- 生成可能終了日
- 生成可能曜日
- 生成受付開始時刻
- 生成受付終了時刻
- 期間内の1ユーザー最大生成依頼数
- 1ユーザー1日の最大依頼数
- 1依頼あたり最大生成枚数

## 注意点

受付方式を `受付時間内なら生成可能` にした場合、教室予約や出席確認がなくても、設定した受付時間内であれば生成申請できます。

教室参加者だけに限定したい場合は、`生成申請の受付方式` を `教室参加者のみ` にしてください。

期間限定キャンペーンでは、開始日と終了日を必ず設定してください。終了日を過ぎると受付できなくなります。
