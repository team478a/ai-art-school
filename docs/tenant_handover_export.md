# v5.2.5 クライアント別 引き継ぎメモ出力

## 目的

新規クライアント導入時に、LINE、LIFF、Stripe、AI、公開ページ、運用ルールの設定状況を1つのMarkdownファイルとして出力できるようにします。

## 追加内容

- `/admin/tenants/{id}/handover` を追加
- クライアント別設定画面に「引き継ぎメモを出力」ボタンを追加
- 以下をMarkdownで出力
  - クライアント情報
  - LINE / Stripe / LIFF の専用URL
  - 初期設定チェック結果
  - 設定値一覧
  - 導入時の確認事項

## セキュリティ

APIキー、Channel Access Token、Webhook Secretなどの秘密情報は実値を出力せず、「設定済み（非表示）」として出力します。

## 使い方

1. `/admin/tenants` を開く
2. 対象クライアントの「設定」を開く
3. 「引き継ぎメモを出力」を押す
4. ダウンロードされたMarkdownを導入記録・確認資料として保存する
