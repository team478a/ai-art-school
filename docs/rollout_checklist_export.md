# v5.0.1 横展開 検収チェックリスト書き出し

## 目的

横展開時に、導入設定の完了状況と公開前チェックをMarkdownで書き出せるようにします。

## 追加内容

- `/admin/client-setup/checklist` を追加
- 「横展開・初期設定」画面に「検収チェックリストを書き出す」ボタンを追加
- 初期設定チェック、公開前ヘルスチェック、動作確認、上書き禁止領域をMarkdownで出力
- 横展開画面の文字化けを修正

## 使い方

1. 管理画面で「横展開・初期設定」を開く
2. 必要な設定を保存する
3. 「検収チェックリストを書き出す」を押す
4. ダウンロードされたMarkdownを導入記録・納品前確認として使う
