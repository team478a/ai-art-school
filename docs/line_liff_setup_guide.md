# LINE公式ID・予約用LIFF ID・購入用LIFF ID 設定手順

この手順書は、新しいクライアントを追加したあとに設定するLINE関連項目の確認用です。

管理画面の入力場所:

- 管理画面 → クライアント管理 → 対象クライアント → 設定
- 「LINE・LIFF」欄に以下を入力します。

## 1. LINE公式ID

LINE公式IDは、友だち追加QRや案内文に使う公式アカウントのIDです。

1. LINE Official Account Managerにログインします。
2. 対象の公式アカウントを開きます。
3. 「設定」→「アカウント設定」を開きます。
4. 「ベーシックID」または「プレミアムID」を確認します。
5. 管理画面の「LINE公式アカウントID」に入力します。

入力例:

```text
@386ipbjr
```

注意:

- 先頭の `@` も含めて入力してください。
- 表示名ではなく、IDを入力してください。

## 2. 予約用LIFF ID

予約用LIFF IDは、LINE内で予約カレンダーを開くために使います。

1. LINE Developersにログインします。
2. 対象プロバイダーを開きます。
3. LINE Loginチャネルを作成、または既存のLINE Loginチャネルを開きます。
4. 「LIFF」タブを開き、「追加」を押します。
5. LIFFアプリ名を `予約カレンダー` などにします。
6. サイズは `Full` または `Tall` を選びます。
7. エンドポイントURLに、管理画面に表示される「予約カレンダー LIFF URL」を入力します。
8. Scopeは `profile` と `openid` を有効にします。
9. Botリンクで対象のLINE公式アカウントを関連付けます。
10. 作成後に表示されるLIFF IDを、管理画面の「予約用LIFF ID」に入力します。

LIFF IDの例:

```text
2000000000-AbCdEfGh
```

注意:

- URLではなく、LIFF IDを入力します。
- エンドポイントURLには、クライアント専用の予約カレンダーURLを設定してください。

## 3. 購入用LIFF ID

購入用LIFF IDは、LINE内で回数券、月額サブスク、年額サブスク、一回払いなどの購入ページを開くために使います。

1. 予約用と同じLINE Loginチャネルを開きます。
2. 「LIFF」タブから、もう一つLIFFアプリを追加します。
3. LIFFアプリ名を `購入・会員メニュー` などにします。
4. サイズは `Full` または `Tall` を選びます。
5. エンドポイントURLに、管理画面に表示される「購入ページ LIFF URL」を入力します。
6. Scopeは `profile` と `openid` を有効にします。
7. Botリンクで対象のLINE公式アカウントを関連付けます。
8. 作成後に表示されるLIFF IDを、管理画面の「購入用LIFF ID」に入力します。

予約用と購入用は、別々のLIFFアプリとして作ることを推奨します。

理由:

- 予約ページと購入ページのURLを分けて管理できる
- リッチメニューのボタン設定が分かりやすい
- 不具合時に原因を切り分けやすい

## 4. リッチメニューで使うURL

リッチメニューのボタンは、LIFF URLを開く設定にします。

予約ボタン:

```text
https://liff.line.me/予約用LIFF_ID
```

購入ボタン:

```text
https://liff.line.me/購入用LIFF_ID
```

管理画面上でLIFF IDを登録している場合は、システム側で必要なリンクを生成できます。

## 5. 動作確認

設定後、以下を確認してください。

1. 管理画面 → クライアント管理 → 対象クライアント → 疎通チェックを開く
2. LINE公式ID、予約用LIFF ID、購入用LIFF IDが未設定になっていないことを確認する
3. LINEアプリ内で予約ページを開く
4. LINEアプリ内で購入ページを開く
5. 予約、購入、LINE認証が正常に動くことを確認する

## 6. よくある設定ミス

- LINE公式IDに `@` が入っていない
- LIFF ID欄にURLを入力している
- 予約用LIFFと購入用LIFFのエンドポイントURLが同じになっている
- エンドポイントURLに別クライアントのURLを設定している
- LINE LoginチャネルとMessaging APIチャネルの公式アカウント連携ができていない
- LINE Loginチャネルが開発中のままで、開発者以外が認証できない

## 7. 新規クライアント追加時の目安

最低限、以下の3つを設定すると、LINE導線の初期確認ができます。

- LINE公式ID
- 予約用LIFF ID
- 購入用LIFF ID

その後、Messaging APIのChannel Secret、Channel Access Token、Stripe、AI APIを設定してください。
