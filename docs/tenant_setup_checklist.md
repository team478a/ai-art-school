# v5.2.4 クライアント別 初期設定チェック

## 目的

SaaS運用で、クライアントごとの設定漏れを一覧で確認できるようにします。

## 追加内容

- `TenantSetupChecklistService` を追加
- `/admin/tenants` に「クライアント別 初期設定チェック」を追加
- 以下のカテゴリ別に未設定項目を表示
  - 基本情報
  - 公開ページ・法務表記
  - LINE・LIFF
  - Stripe・料金
  - AI生成
  - 運用ルール
- クライアントごとの設定完了率を表示

## 使い方

1. `/admin/tenants` を開く
2. 「クライアント別 初期設定チェック」を確認する
3. 未設定があるクライアントは「設定を確認」を押す
4. クライアント別設定画面で不足項目を入力する

## 補足

このチェックは運用開始前の確認用です。API接続の実通信テストではなく、必要な設定値が登録されているかを確認します。
