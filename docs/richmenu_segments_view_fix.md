## v5.4.4 リッチメニュー設定画面の表示修正

### 修正内容

- `/admin/richmenu-segments` が途中で表示停止し、共通管理画面レイアウトが読み込まれない問題を修正しました。
- 原因だった未定義の `h()` 関数呼び出しを、画面内の安全なエスケープ処理に置き換えました。
- 管理画面の既存CSSクラスに合わせて、カード、ボタン、通知表示を整えました。
- 現在操作中のクライアント名とキーを画面上部に表示し、この設定が対象クライアント専用であることを明示しました。

### 確認ポイント

1. `/admin/richmenu-segments` を開く
2. 左メニュー、上部バー、現在操作中クライアント表示が出る
3. 初回・参加済み・回数券・サブスクのリッチメニューID入力欄が表示される
4. 保存と同期ボタンが表示される
