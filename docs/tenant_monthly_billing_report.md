# テナント別 月次利用・請求レポート

## 目的

SaaS運用で、オーナーがクライアントごとの利用量と売上を月単位で確認できるようにします。

## 追加画面

- `/admin/tenants/monthly-report`

オーナー権限のみ表示できます。

## 集計対象

- 売上
- 返金
- 差引売上
- 決済件数
- 予約件数
- 承認件数
- 参加件数
- 画像生成成功件数
- 画像生成失敗件数
- LINE送信数

## 注意

Stripe側の確定売上と差がある場合は、Webhook未反映、返金、現金決済、テーブル構造の違いを確認します。
この画面はシステム内ログをもとにした運用確認用です。
